== Installation ==

1. Install the plugin through the WordPress plugins screen directly, or upload the plugin files to the `/wp-content/plugins/ultimate-loan-mortgage-calculator` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Choose the Dashboard->Loan & Mortgage menu item to configure the plugin.
4. Place the shortcode `[ultimate_loan_mortgage_calculator]` in your pages or posts where you want the calculator to appear.
